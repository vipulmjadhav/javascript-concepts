DOM - Introduction 

        DOM :It represents the structure of web pages.  
        Document object Manipulation is not a language feature . It is independant feature
        and any language can use it.

        1.language can access , Manipulate documents using DOM as DOM Controls the structure of document
        documents: It is a simple web page or a html source.

        2.IN Document unit of control is node ie everything is in terms of node and the DOM has access to
        the structure of Node , hence JS without DOM cant access web pages.

        3.In other words DOM is the collection of node, which provides other language a functionality 
        to access this node structure.

        4.Using DOM a language can:
            add/remove html elements.
            access/modify html elements
            style html elements.

            Type of Nodes : textNode, Attribute Node, elementNode.
